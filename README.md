# LovenseRBLX
Control your Lovense toy with Roblox

> loadstring:

```lua
local Lovense = loadstring(game:HttpGet('https://raw.githubusercontent.com/esexual/LovenseRBLX/main/Lovense.lua'))()
```

```lua
local Lovense = loadstring(game:HttpGet('https://raw.githubusercontent.com/esexual/LovenseRBLX/main/Lovense.lua'))()
Lovense.GetToys()

Lovense.Vibrate(10,5);
```
